# Tiny MC - 项目主页

一个基于Web的游戏项目主页，展示Tiny MC游戏的特色和截图。

## 🎮 项目简介

Tiny MC是一款像素风格的沙盒建造游戏，玩家可以在无限的世界中自由建造、探索和生存。

## ✨ 功能特性

### 页面功能
- **导航系统** - 平滑滚动导航，支持跳转到各个区域
- **游戏特色展示** - 6个特色卡片展示游戏核心玩法
- **截图画廊** - 可切换的游戏截图展示，支持键盘控制
- **页脚信息** - 包含关于我们、联系我们、隐私政策、服务条款等信息

### 动效系统
- **鼠标轨迹** - 彩虹渐变的鼠标移动轨迹效果
- **点击特效** - 丰富的粒子爆炸效果
- **呼吸灯背景** - 动态颜色切换的背景光效
- **滚动动画** - 元素进入/离开视口时的渐显/渐隐效果
- **按钮交互** - 悬停和点击的缩放反馈动画

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计与动画
- **JavaScript** - 交互逻辑与动画控制
- **Tailwind CSS** - 响应式布局框架

## 📁 项目结构

tinymc home/

## 🚀 快速开始

1. **克隆项目**
```bash
git clone https://github.com/tiny-mc/TinyMCHome/clgon.git
cd TinyMCHome
```

2. **启动开发服务器**
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve
```

3. **访问页面**
打开浏览器访问 `http://localhost:8000`

## 🎯 使用说明

### 导航
- 点击顶部导航栏的"首页"、"特色"、"截图"链接可平滑滚动到对应区域
- 点击"游玩"链接跳转到游戏页面

### 截图画廊
- 点击缩略图切换图片
- 使用左右箭头按钮切换
- 使用键盘左右方向键切换

### 交互效果
- 移动鼠标查看轨迹效果
- 点击页面任意位置查看点击特效

## 📱 响应式设计

页面支持多种设备尺寸：
- 桌面端（1200px+）
- 平板端（768px-1199px）
- 移动端（<768px）
